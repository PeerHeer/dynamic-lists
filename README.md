# NBT List
An abstraction for operations on NBT lists.

### Compatibility information
- Namespace: `nbtlist`
- Scorespace: `nbtlist`
- Incompatible with: `none`

### Planned features
- Partial data lookups (e.g.`List[{ID: 0}}` returns `Data: {ID: 0, Tag: "something"}`)
- Shuffle
- Error codes and error messages
